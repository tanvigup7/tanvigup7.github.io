I swear, the old photograph glows gold, Edges softened, time blurred. The colors, too rich, too vivid More than memory should allow. I swear, the sky used to be bluer, Laughter rang fuller, echoing deep. Silence hummed with something more, streets alive with motion. More than memory should allow. The present drags, more than memory should allow. Muted and slow, grayer. Footsteps vanish into hollow air. I swear, my voice is fading. The world is darkened, more than memory should allow. The sun is long set, dim the lights, The doors to the past, all rusted and sealed. I swear, the past still burns so bright.
